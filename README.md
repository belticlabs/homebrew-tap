# Homebrew Tap for Beltic

This is the official Homebrew tap for [Beltic CLI](https://github.com/belticlabs/beltic-cli).

## Installation

```bash
brew tap belticlabs/tap
brew install beltic
```

Or in a single command:

```bash
brew install belticlabs/tap/beltic
```

## Update

```bash
brew upgrade beltic
```

## Uninstall

```bash
brew uninstall beltic
brew untap belticlabs/tap
```

## What is Beltic?

Beltic is a CLI for managing and signing verifiable credentials for AI agents.

- Generate Ed25519 keypairs
- Create and sign developer/agent credentials
- Sign HTTP requests with RFC 9421 message signatures
- Generate key directories for Web Bot Auth
